Función del XML:
La función del XML es estructurar y almacenar datos de forma ordenada y legible. Lo usamos para configurar, transportar o 
describir datos, no para mostrarlos visualmente.

Función del HTML:
La función del HMTL en el proyecto es definir la estructura y el contenido de la página web. Nos permite organizar el texto, 
las imágenes, los enlaces y otros elementos visibles en el navegador.
